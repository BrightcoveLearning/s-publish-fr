Sur la plupart des plateformes, l'intégration de vidéos directement dans un e-mail n'est pas possible.™ Il ne s'agit pas d'une limitation de Brightcove, presque tous les systèmes de messagerie modernes suppriment le code intégré du lecteur vidéo pour des raisons de sécurité. Vous pouvez toutefois augmenter la notoriété et le trafic du contenu en créant un e-mail avec un texte d'accroche et une image d'affiche. Le module Médias permet de publier une vidéo sur une page de destination d'e-mail.

Remarque: Pour une autre option d'envoi de vidéos par e-mail qui permet de mieux contrôler l'apparence de l'image de l'affiche de la vidéo et utilise une page de destination personnalisée, voir Pilotage des vues vidéo par e-mail.

Pour publier une vidéo sur la page d'accueil d'un e-mail, procédez comme suit:

  1. Ouvrez le module Médias.
  2. Sélectionnez une vidéo.
  3. Cliquez sur Publier et intégrer > Page de destination de l'e-mail.
  4. Sélectionnez une page de destination. La liste déroulante contiendra une page de destination simple qui est un exemple de page de destination sans style fourni par le système. La liste contiendra également toutes les expériences de page de destination publiées qui ont été créées à l'aide du module Gallery. L'utilisation d'une page de destination Gallery offre plus de flexibilité quant à l'apparence et au style de la page de destination. Les formulaires de prospects peuvent également être utilisés pour capturer des informations sur les spectateurs. Pour plus d'informations sur la création de pages de destination de la Galerie, consultez Créer une expérience de page de destination.
    Remarque: L'image de l'affiche sera dimensionnée à 640 x 360.
  5. Sélectionnez cette option pour lire automatiquement la vidéo. Cette option n'est disponible que lorsque vous sélectionnez la page de destination simple.
  6. Sélectionnez un lecteur vidéo. Cette option n'est disponible que lorsque vous sélectionnez la page de destination simple. Pour les pages de destination de la Galerie, le lecteur est assigné dans le cadre des propriétés de l'expérience.
  7. (Facultatif) Cliquez sur le lien Aperçu de la page de destination pour prévisualiser la page de destination.
  8. Mettez en surbrillance l'image, copiez-la dans le presse-papiers, puis collez-la dans un e-mail.

Cliquez sur Générer un lien court pour générer un lien raccourci qui peut également être utilisé et distribué. Lorsque l'e-mail est reçu, cliquez sur l'image de l'affiche ou sur le lien court pour ouvrir la vidéo dans un onglet du navigateur.

Si une page de destination Gallery est utilisée avec un formulaire de prospect, les données du formulaire de prospect peuvent être téléchargées à l'aide du module Campaign. Pour plus d'informations, consultez la section Exportation des données des prospects
