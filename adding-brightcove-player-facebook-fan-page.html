L'exemple ci-dessous montre un lecteur intégré à une page de fans Facebook.

Installation de l'application HTML statique

Le processus décrit ci-dessous utilise une application que vous ajoutez à votre compte Facebook. Cette application créera un onglet sur votre page Facebook où vous pourrez ensuite coller du code HTML statique affiché sur cet onglet. Notez qu'il existe de nombreuses autres applications qui fournissent cette fonctionnalité. Les étapes ci-dessous supposent que vous disposez d'une page de fans Facebook. Si ce n'est pas le cas, vous pouvez aller ici pour en créer un.

  1. Ajoutez l'application Static HTML: iFrame tabs-fr/ en tant qu'onglet sur votre Fan Page. Pour ajouter cette application, rendez-vous sur https://apps.facebook.com/static_html_plus. Suivez les instructions pour installer l'application sur votre compte.
  2. Après avoir installé l'application, vous devriez voir un onglet Bienvenue sur votre page de fans. Cliquez sur l'onglet Bienvenue.
  3. Cliquez sur l'onglet Configuration. Cela ouvrira un éditeur dans lequel vous pourrez coller du HTML.
  4. Dans {{site.product}}, copiez le code intégré Standard (iframe) du lecteur que vous souhaitez ajouter à votre Fan Page Facebook.
  5. Collez le code intégré dans l'éditeur HTML statique.
  6. Cliquez sur Aperçu pour vérifier que votre lecteur s'affiche correctement.
  7. Si l'aperçu était correct, cliquez sur Enregistrer et publier pour enregistrer vos modifications.
  8. Pour voir le lecteur, ouvrez votre page Facebook et cliquez sur l'onglet Bienvenue.

Modification du nom de l'onglet

Vous pouvez modifier le nom de l'onglet Bienvenue en un nom plus descriptif en suivant les étapes ci-dessous.

  1. Connectez-vous à votre compte Facebook.
  2. Ouvrez votre page Facebook sur laquelle vous avez ajouté le lecteur.
  3. Cliquez sur le lien Paramètres en haut de la page.
  4. Cliquez sur Modifier la page dans le volet de navigation de gauche.
  5. Localisez l'onglet Bienvenue, puis cliquez sur Paramètres.
  6. Cliquez sur Modifier les paramètres.
  7. Une boîte de dialogue s'ouvre pour vous permettre de renommer l'onglet. Il existe également des options permettant de le supprimer et de modifier l'icône associée à l'onglet. Saisissez un nouveau nom pour l'onglet.
  8. Cliquez sur Enregistrer, puis sur OK.
  9. Cliquez sur Enregistrer. Notez que l'interface utilisateur des paramètres de Facebook affichera toujours le nom d'onglet Bienvenue, mais le nouveau nom apparaîtra lorsque vous afficherez la page.

Modification du code HTML

Si vous devez modifier le code HTML associé à l'onglet, procédez comme suit.

  1. Connectez-vous à votre compte Facebook et rendez-vous sur votre page.
  2. Cliquez sur l'onglet qui représente la page de votre vidéo, puis cliquez sur l'onglet Modifier.
  3. Cliquez sur le code HTML comme il convient. Cliquez sur Enregistrer et publier lorsque vous avez terminé.
