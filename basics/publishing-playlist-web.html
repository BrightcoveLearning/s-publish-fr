Prerequisites

  • You have uploaded videos to your Video Cloud account
  • You have created a Manual or Smart playlist

Steps

  1. In the primary navigation, open the Media module.
  2. Select a playlist to publish:
    • If the playlist appears in the side navigation, click it and then click Publish Playlist
    • Click All Playlists in the left navigation menu, select a playlist and then click Publish and embed... Ë‡ > Publish to Web
  3. In the Select Player dropdown list, select a player to use.
  4. Click the URL below the preview player to open it in a new browser tab. Confirm that the playlist appears and that the videos can be played.
  5. Copy the embed code and paste it into your webpage.
    Note: If the selected player has been configured with the Campaign plugin, the iFrame code will not be displayed as the JavaScript code must be used.
Note: For complete details on publishing playlists to the web, see Publishing Videos and Playlists with the Media Module.
