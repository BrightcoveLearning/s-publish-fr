Brightcove fournit également le plugin WordPress Brightcove Video Connect. Ce plugin prend en charge plusieurs comptes Brightcove et vous permet de télécharger des vidéos, de les ajouter à des listes de lecteur et d'afficher des shortcodes avec vos vidéos, le tout depuis l'interface d'administration de WordPress.

Étapes pour publier un lecteur Brightcove sur une page WordPress

Les étapes et les captures d'écran ont été effectuées à l'aide de WordPress 5.2.2.

  1. Connectez-vous à votre site WordPress et allez dans Posts dans la barre latérale gauche.
  2. Cliquez sur Ajouter nouveau pour ajouter une nouvelle publication.
  3. Ajoutez un titre à la publication.
  4. Cliquez sur le signe + pour ajouter un bloc. Dans la section Formatage, cliquez sur HTML personnalisé pour ajouter un bloc HTML.
  5. Collez le code intégré de l'iframe dans l'éditeur. Modifiez les paramètres de largeur et de hauteur selon vos besoins.
  6. Cliquez sur Aperçu pour prévisualiser le lecteur.
  7. Cliquez sur Publier... Vérifiez les paramètres de publication, puis cliquez sur Publier.
  8. Pour voir le lecteur sur votre site, cliquez sur le bouton Afficher la publication sur le côté droit de la page.
  9. Si vous apportez des modifications au code de publication ou au titre, assurez-vous de cliquer sur le bouton Mettre à jour pour enregistrer vos modifications.
