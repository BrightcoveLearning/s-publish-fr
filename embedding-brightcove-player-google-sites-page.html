Google Sites est un outil de création de pages Web proposé par Google. Les lecteurs Brightcove peuvent facilement être intégrés aux pages Google Sites.

Pour intégrer un lecteur Brightcove sur une page Google Sites, procédez comme suit:

  1. Localisez la page Google Sites à laquelle vous souhaitez ajouter un lecteur Brightcove.
  2. Cliquez sur le bouton Modifier la page.
  3. Une barre d'outils apparaît en haut de la page. Cliquez sur le bouton <HTML>.
  4. Collez le code de publication de votre lecteur Standard (iframe) et cliquez sur Mettre à jour.
  5. La page sera mise à jour et affichera un gadget Google pour l'iframe.
  6. Cliquez sur Enregistrer pour enregistrer les modifications. La vidéo sera affichée sur la page.
